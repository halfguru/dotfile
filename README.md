## ✨ Features

- 🪟 **Hyprland** - Tiling Wayland compositor with custom keybinds
- 📝 **Neovim** - LazyVim-based config with Omarchy theme integration
- 🐚 **Zsh** - Oh-my-zsh with autosuggestions & syntax highlighting
- 🤖 **Openencode** - AI assistant configuration

## 🚀 Install

```bash
git clone https://github.com/simondkho/dotfiles.git ~/.dotfiles
cd ~/.dotfiles
./install
```

> ⚠️ Existing configs are backed up with `.bak` extension

## ⌨️ Key Bindings

### Hyprland

| Binding | Action |
|---------|--------|
| `Super + Return` | Terminal |
| `Super + Alt + Return` | Tmux |
| `Super + Shift + F` | File manager |
| `Super + Shift + N` | Editor (Neovim) |
| `Super + Shift + B` | Browser |

### Zsh Aliases

| Alias | Command |
|-------|---------|
| `ls` | `eza --icons` |
| `ll` | `eza -l --icons` |
| `cat` | `bat` |
| `gs` | `git status` |
| `gc` | `git commit` |
